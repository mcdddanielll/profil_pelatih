## 📝 Customisasi

### Mengganti Foto Pelatih
Untuk menambahkan foto asli pelatih, ganti background pada class berikut di `styles.css`:

```css
.coach-bayu {
    background: url('path/to/coach-bayu.jpg');
    background-size: cover;
    background-position: center;
}
```

### Menambah Galeri
Tambahkan foto di section gallery dengan mengganti:

```html
<div class="gallery-item">
    <img src="path/to/image.jpg" alt="Description">
</div>
```

### Mengubah Warna Tema
Edit variabel CSS di bagian `:root` pada `styles.css`:

```css
:root {
    --primary-blue: #0066CC;
    --orange: #FF6B35;
    /* ... */
}
```
